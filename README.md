Weather Website:
  Website that allows users to see real time weather alerts as well as the current temperature, airspeed, precipitation, forecast, and temperature for the next week. 
  Created for HackUO 2024.

Libraries used:
  streamlit
  pandas
  folium
  streamlit_folium
